# 绩效管理系统 Demo

基于百人公司场景的完整绩效管理系统演示。

## 快速启动

```bash
# 需要 Node.js 22+
npm install
npm start

# 访问 http://localhost:3000
```

## 功能模块

- **首页仪表盘** - 角色差异化展示，公司/团队/个人概览
- **目标管理** - OKR（目标+关键结果）和 KPI 双模式，进度跟踪
- **绩效评估** - 员工自评（5维度评分）+ 经理评审
- **360反馈** - 匿名/实名反馈，优势/改进双维度
- **绩效校准** - 团队等级分布可视化，强制分布参考
- **数据报表** - 评级分布图、部门对比、员工排名（Chart.js）
- **员工管理** - 100人员工库，增删改查（HR专属）
- **周期管理** - 创建/推进绩效周期状态（HR专属）
